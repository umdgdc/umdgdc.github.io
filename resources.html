Student Resources

This page is an aggregate list of online tutorials and web resources that the club has put together in order to help new members break in to and understand game development. Click on the dropdown topics below to display tutorials and information relating to the described topics.

Roll a Ball

Description: Roll a Ball is a standard intro Unity tutorial, it will walk you through the basic functionality of the unity editor and teach you about the basics of unity's physics/collision system and c# scripting. This is a good starting point for learning how the unity editor and workflow works. This will probably take you an hour or two if you are new to unity.

Covers: Physics, Character Movement, Basic Scripting

Brackey's Unity Intro Tutorial

Description: This covers mostly the same material that roll a ball does but the game it makes is different. The reason we included this video series is because it is by Brackeys, the most prolific Unity Tutorial Youtuber. Brackeys has made videos covering many of the key features of Unity, so if you watch this tutorial and can follow along he is an excellent resource for beginner devs.

Covers: Sprite Sheets, Camera Movement, Character Movement, and Physics

Making a Character Controller with Custom Physics

Description: This video series shows how you can use some of Unity's components to customize and reimplement parts of their physics system, in order to create a more responsive and predictable 2D character controller. I would reccomend this tutorial if you want to make a game where physical interactions matter, or if you are unhappy with how your character controls while using Unity's standard physics and collision detection. This tutorial is slightly more advanced than the first two, but is very useful.

Covers: Physics,Collision Detection, Lots of Scripting

2d Animations

Description: This tutorial will cover how to extract animations out of sprite sheets and turn them into Unity animation files. It then shows how to apply those animations to a basic 2d character controller in order to play the animations at the correct times.

Covers: Animation, Character Movement, Scripting

3d Animations

Description: This is a Unity live training session that covers how to rig 3d Animations to 3d models. Rigging animations means applying a 3D animation file to a so it can manipulate the models appearences at the correct times. The session is a bit lengthy, but the real tutorial doesn't start until 18 minutes in so you can skip that if you want to. Further down the page, in the blender section, there are links detailing how to make your own 3d animations instead of using the stock asset used in this tutorial.

Covers: Animation, Rigging, Scripting

Unity Navmesh

Description: A navmesh is a surface that gameobjects can use for pathfinding and determining the most efficent way to reach a designated position. This is a very convenient way to handle AI movement and pathfinding in Unity. This tutorial series covers how to make and use a Navmesh and later shows how to make Navmeshes at runtime, which is very useful if you want to make a procedurally generated game.

Covers: Navmeshes and the Navmesh Component

Basic NPC Follow Script

Description: This tutorial shows how to make a very basic AI Gameobject that faces and follows the player. It also covers how to rig this agent.

Covers: AI, Rigging

UNET LAN Multiplayer

Description: This tutorial covers how to use Unity's built-in networking system to make a multiplayer game that will work when two instances of it are open on the same computer or network. The first video shows how to set up the networking components and scene and the second shows how to find games over LAN. I would not reccomend using UNET for online matchmaking or gameplay, Photon Networking is a far superior tool in that regard, and it is covered below.

Covers: UNET, Network Discovery, Networking Concepts

Local Multiplayer with Multiple Gamepads/Keyboards/Mice

Description: This videos will show you how to set up an effective input management system for recieving and managing inputs from multiple input devices. To achieve this we use the popular Unity asset Rewired, which the club Unity account has a paid copy of. You can use a free trial of this asset or talk to the club president at a meeting if you want to recieve access to the club Unity account and assets. To actually use this input management system in a game you should read through THIS documentation and replicate what they do for the character controller.

Covers: Input Management

Setting up the Input Manager

Setting Up and Accessing an Online Database

Description: This video series covers how to set up an online sql database through phpmyadmin and how to use PHP to let Unity edit and access this database. This can be very useful for doing simple online login, high score lists, and cloud saves in Unity.

Covers: PHP, SQL, Databases

Online Multiplayer Using Photon

Description: Photon is a networking framework similar to Unity's built in networking. Like the rewired asset, the club has a paid version of this asset available, however you will not need the paid version unless you actually publish your game or your game exceeds 100 current online users/sessions, so the trial should be fine for demos and testing.
